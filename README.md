# StyleIQ

Universal fashion recommendation engine built using Next.js and TypeScript.

## Tech Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- PostgreSQL (Supabase - upcoming)
- GitHub Collaboration Workflow

## Project Vision
StyleIQ aims to provide personalized fashion recommendations based on user inputs like height, weight, occasion, and preferences.

## Development Workflow
- main → stable branch
- backend-dev → backend development
- frontend-dev → frontend development
